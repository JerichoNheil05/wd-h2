Hands on 2

This project features a modern web design that includes a striking hero section and a card gallery showcasing a blend of animated and static cards. The hero section captures the viewer's attention immediately, while the card gallery offers a visually engaging and interactive experience.This project aims to showcase the potential of combining static and dynamic elements in a web design, providing a balanced and engaging user experience.
